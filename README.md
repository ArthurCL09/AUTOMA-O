🤖 Automação de Cadastro de Produtos

Projeto de automação desenvolvido para cadastrar produtos de forma rápida e eficiente em sistemas/websites, eliminando tarefas repetitivas e reduzindo erros manuais.

🚀 Funcionalidades
Cadastro automático de produtos
Preenchimento de formulários (nome, preço, descrição, etc.)
Upload automático de imagens
Integração com planilhas (Excel/CSV)
Simulação de ações do usuário (clique, digitação, navegação)
🛠️ Tecnologias utilizadas
Python
PyAutoGUI
Pandas
